Getting started with Hipchat

Atlassian's HipChat now has free group chat - we'll use it to communicate between meetings. You can use this sign up link to join the Code Sydney group.

Use HipChat to ask questions, offer support, and stay in contact between meetings. Try not to give away the answer, but offer hints and tips - we're all here to learn!

As always, leave a comment on this post or contact me if you have any problems getting started.
